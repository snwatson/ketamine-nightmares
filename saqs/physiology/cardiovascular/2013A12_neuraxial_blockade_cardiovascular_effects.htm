Intro: subarachnoid or epidural local anaesthetic

Summary of effects

·  Insult ∝ block height, local anaesthetic concentration and volume

·  All changes lead to hypotension

o Vasodilation -> ↓SVR -> ↓mAP

o Venodilation -> ↓venous return, ↓cardiac output, ↓mAP

o ±Myocardial depression -> ↓cardiac output, ↓mAP

o Impaired neurohormonal compensation

·  Especially dangerous if

o Hypovolaemic (e.g. dehydration in prolonged labour)

o Fixed cardiac output (e.g. aortic stenosis)

o Susceptible to LVOT obstruction (e.g. HOCM)

Neural compensation

·  Baroreceptor reflex:

o Stretch-sensitive mechanoceptors in carotid and aortic sinuses

o ↓Stretch -> ↓afferent to medulla -> ↑SNS and ↓PSNS output

o ↑HR, vaso-/venoconstriction

Hormonal compensation

·  ↑Renin-angiotensin-aldosterone system activation

o Vasoconstriction

o ↑Na+/H2O reabsorption -> ↑blood volume -> ↑venous return

·  ↑ADH release

o Vasoconstriction

o ↑H2O reabsorption -> ↑blood volume -> ↑venous return

·  ↓ANP release

o ↓Natriuresis -> ↑blood volume -> ↑venous return

 

Cardiovascular innervation:

 

SNS

PSNS

Pre-motor

·  Rostral ventrolateral medulla

N/A

Pre-ganglionic neuron

·  Lateral horn
(releases ACh)

·  Dorsal motor nucleus of CNX

·  Nucleus ambiguous
(releases ACh)

Post-ganglionic neuron

·  SNS paravertebral ganglia
(releases NAd)

·  Local ganglia at target organ
(releases ACh)

Receptor and effect

·  Adrenergic

·  α1: vasoconstriction -> ↑SVR

·  α1: venoconstriction -> ↑venous return

·  β1 -> ↑HR, ↑contractility -> ↑cardiac output

·  Muscarinic

·  m2AChR: ↓HR, ↔/↓contractility -> ↓cardiac output

·  Minimal vascular effect

Spinal cord outflow

·  Upper thoracic: head, upper limbs, thorax
(heart T1-4)

·  Mid-lower thoracic: most of abdomen
(kidneys T10-L2)

·  Upper lumbar: most of pelvis, lower limbs

·  CN III, VII, IX -> head and neck

·  CN X -> upper limbs, thorax, most of abdomen

·  Sacral -> lower abdomen, pelvis, lower limbs

 

Table: effects by block height

Lumbar

·  Minimal effect

Thoracic

·  ↓SVR and ↓venous return – lower body

·  Mild ↓mAP ∝ height

Cervical

·  ↓SVR and ↓venous return – whole body

·  ↓HR, ↓contractility

·  Significant ↓mAP

Brainstem

·  ↓SVR and ↓venous return – whole body

·  ↓HR, ←contractility

·  Profound ↓mAP

(I’m not so sure about the effects of brainstem block)

 

Compensatory effects by block height:

Lumbar

·  Minimal need

Thoracic

·  Baroreceptor response

o ↑HR

o ↑SVR and venous return – upper body

·  ADH release

·  RAAS activation by renal (↓perfusion, ↓GFR) and central (SNS) mechanisms

·  ↓ANP release

Cervical

·  ADH release

·  RAAS activation by renal mechanisms

·  ↓ANP release

Brainstem

·  No central response; HR at intrinsic SA node rate ~100-110

·  RAAS activation by renal mechanisms

·  ↓ANP release
