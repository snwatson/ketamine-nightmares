Obesity

 

Disease:

Classification

BMI:

·     Class 1: >30

·     Class 2: >35

·     Class 3: >40

Others:

·     Neck circ: >43cm male, >41cm female – important for breathing

·     Waist circ: >102cm male, >83cm female – important for metabolic

Weight types

·     Total = measured

·     Ideal = (height – 100) if male, (height – 105) if female

·     Lean = fancy equations, limit 100kg if male, 70kg if female

·     Adjusted = ideal + 40% of excess

Problems

·     Obesity itself -> A, B, C

·     Sleep-disordered breathing -> A, B, C

o  Especially if undiagnosed/untreated

·     Metabolic syndrome: -> all things inflammatory

o  Features: obesity, dyslipidaemia, hypertension, hyperglycaemia

o  Main risks: AMI, VTE, SSI

·     Obesity paradox

o  ↑Morbidity: dose-dependent

o  ↓↑Mortality: nadir at Class 2 (!)

 

Airway:

Difficulty

Problems:

·     ↓Cavity

·     ↓Positioning

·     ↓Time

Mitigation:

·     Ultimate: awake intubation

·     Planning: A, B, C, D

·     Assistance: nurse + anaesthetist/trainee

·     Positioning: Oxford ramp + head cushion vs jack knife table vs pillows

o  ↑axis alignment, ↑FRC, ↑compliance

o  Tragus level with sternal angle

·     Pre-oxygenation: etO2 >90%, aim SpO2 >92% for >3 mins

·     Apnoeic oxygenation: HFNP

·     Induction: mRSI

·     Bag-mask ventilation: two hands + guedel

·     Video laryngoscope + hyperangulated blade + ETT with rigid stylet

Aspiration

Problems:

·     ↑Intra-abdominal pressure (mechanical pressure)

·     ↓Emptying (? T2DM -> gastroparesis)

·     ↑Volume (large meals)

·     Sphincter incompetence (GORD, hiatus hernia)

·     Insufflation (difficult bag-mask ventilation)

Mitigation:

·     Secure: always intubate

·     ↓Food: light meal, 6h fast, prokinetic

·     ↓Air: no ventilation or two hands + guedel

·     ↓Acid: H2A, PPI, sodibic

·     ↓Time: mRSI

Awareness

Problem:

·     ↑Cardiac output -> ↑distribution

·     ↑Difficulty -> ↑delay

Mitigation:

·     Propofol TCI vs turn on vapouriser early

·     Fast relaxant (1.2mg/kg roc)

·     Fast opioid (20mcg/kg IBW alf)

Extubation

Pre-requisites:

·      Check ABC and ABG

·      Recruitment manoeuvre

·      Suction upper airway

·      Bite block

·      Reversal: sugammadex + TOF >0.9 confirmation

·      Eyes opening and obeying commands

Extubation:

·      Onto bed via hover mat

·      Sitting up

·      Pull tube after inspiration

·      Straight onto CPAP 10cmH2O

 

Breathing (see OSA)

Oxygenation

Problems:

·     Small airway closure -> shunt -> hypoxaemia

·     Low FRC -> rapid desaturation

Mitigation:

·     PEEP

Ventilation

Problems:

·     Restrictive deficit -> hypoventilation

Mitigation:

·     Lung-protective ventilation

 

Circulation (see OSA)

↑Cardiac output

·     50kg fat = 1L cardiac output

·     ↑Distribution

o   ↑Rate of offset after bolus -> awareness at induction

o   ↑Accumulation during infusion -> slow emergence, obtundation

↑Blood volume

·     But ~50mL/kg

·     ↑Dilution of centrally-distributed drugs -> ↑IV induction doses

Pathology

·     Obesity -> AF, cardiomyopathy

·     Metabolic syndrome -> hypertension, LVH, IHD

·     Sleep-disordered breathing -> pulmonary hypertension, cor pulmonale

 

Drug dosing:

Total

·     Sux

·     LMWH

Adjusted

·     Propofol maintenance

·     Antibiotics

·     Reversal

Lean

·     Propofol induction

·     Opioid

·     Relaxant

·     Local

 

Extra:

Access

·     Long PIVC and long A-line

·     Ultrasound guidance

Monitoring

·     BP: correct size cuff ± forearm application ± A-line instead

·     BIS: for drug accumulation vs awareness

·     NMT: for immobility with minimal opioid + safe extubation

Positioning

·     Many hands

·     Big table + armboards

·     Hovermat

·     Straps

·     Pads

Thromboprophylaxis

·     Early mobilization

·     Calf compressors – correct size

·     Anticoag – correct dose (e.g. 100-150kg -> enoxaparin 40mg bd)
