2019A01 Explain the effects of intermittent positive pressure ventilation on cardiac output.

 

List:

·         Intro

·         Right heart

·         Left heart

·         Indirect effects

·         Hormonal effects

 

Intro:

Determinants of cardiac output

Factors increasing it:

·   ↑Preload (PL)

·   ↓Afterload (AL)

·   ↑Heart rate (HR)

·   ↑Contractility (CY)

Bivalent effect of IPPV

·   ↓LV output if healthy LV

o ↓Preload predominates

·   ↑LV output if failing/overloaded LV

o ↓Transmural pressure predominates

·   Inspiration: ↑LV output, ↓RV output

o ↑Intrathoracic pressure

o Opposite to spont vent

·   Expiration: ↓LV output, ↑RV output

o ↓Intrathoracic pressure

o Opposite to spont vent

Exacerbating factors

↓Preload exaggerated by:

·   PEEP (compression of great veins)

·   Hypovolaemia (↓MSFP, ↓VR)

·   Reverse Trendelenburg (venous pooling)

 

Right ventricle:

 

Inspiration: ↑ITP (intrathoracic pressure)

Expiration: ↓ITP

PL

Early:

-Compress great veins -> emptying -> ↑PL

-Compress RV -> ↓compliance -> ↓PL

Late:

-Compress great veins -> collapse -> ↓PL

-Compress RV -> same

-↑LV vol -> ↓RV compliance -> ↓PL (interdependence)

Other:

-Loss of diaphragm pump -> ↓PL

Early:

-Relax great veins -> pooling -> ↓PL

-Relax RV -> ↑compliance -> ↑PL

Late:

-Relax great veins -> ↑venous return -> ↑PL

-Relax RV -> same

-↓LV vol -> ↑PL (interdep.)

Other:

-Loss of diaphragm pump -> ↓PL

AL

-Compress RV -> ↓transmural pressure -> ↓AL

-Compress lung vessels -> ↑AL

-Relax RV -> ↑AL

-Relax lung vessels -> ↓AL

 

Left ventricle:

 

Inspiration: ↑ITP

Expiration: ↓ITP

PL

Early:

-Compress lung vessels -> emptying -> ↑PL

-Compress LV -> ↓compliance -> ↓PL

Late:

-Compress LV -> ↓compliance -> ↓PL

-Compress lung vessels -> collapse -> ↓PL

-↓RV vol -> ↑LV compliance -> ↑PL (interdep.)

-↓RV output -> ↓LVEDP -> ↓PL

Early:

-Relax lung vessels -> pooling -> ↓PL

-Relax LV -> ↑PL

Late:

-Relax LV -> ↑PL

-Relax lung vessels -> ↑venous return -> ↑PL

-↑RV output -> ↑PL (interdep).

-↑RV vol -> ↓PL

AL

-Compress LV -> ↓transmural pressure -> ↓AL (more important)

-Compress great arteries -> ↑AL (less important)

-Relax LV -> ↑AL

-Relax great arteries -> ↓AL

 

Indirect effects:

Baroreceptor response

·   ↓CO -> ↓mAP -> ↓baroreceptor stretch -> ↑SNS/↓PSNS output

Bainbridge effect

·   ↓PL -> ↓atrial stretch -> ↓HR

Bowditch effect

·   ↑HR -> ↓time for Na+ efflux by Na+K+ATPase -> ↑Ca2+ influx by Na+/Ca2+ exchanger -> ↑CY

Anrep effect

·   ↓AL -> ↓activation of tension-dependent Na+/H+ channel -> ↓Ca2+ extrusion -> ↓CY

 

Hormonal effects:

Renin-angiotensin system

·   Activation:

o ↓mAP -> ↓renal baroreceptor stretch -> ↑renin release

·   Angiotensin effects:

o Peripheral vasoconstriction -> ↑AL

o ↑Na+, H2O reabsorption from prox tubule -> ↑blood volume -> ↑PL

·   Aldosterone effects:

o ↑Na+ reabsorption

o Indirect H2O reabsorption via ↑ECF osmolality -> ↑ADH

·   Net effect

o ↑Blood volume -> ↑PL, ↑CO, ↑mAP

Atrial natriuretic peptide

·   PPV -> ↑CVP -> ↑stretch right atrium -> ↑ANP release

o ↑HR -> ↑CO

o ↑natriuresis
